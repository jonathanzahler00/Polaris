# Polaris

A daily orientation check. Set your intention before the day takes over.

## Features

- 📝 Daily orientation text
- 🔒 Immutable once locked (one per day)
- 📱 Progressive Web App
- 🔔 Push notifications (optional)
- 🎨 Clean, minimal interface

## Development

```bash
npm install
npm run dev
```
